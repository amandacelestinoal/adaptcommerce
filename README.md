# adaptcommerce
trata-se

## requirement

- git
- npm (> = 5.x)
- nodejs (> = 8.x)
- reactjs
- graphql
- mongodb

## installation

1.  `git clone git@github.com:adaptcommerce/adaptcommerce`
2.  `cd adaptcommerce` 
3.  `sudo npm install`
3.  `sudo npm start`
4.   This is it! You can now open http://0.0.0.0:8080/ and begin hacking.

## feature
- free and open
- decentralized
- transparent
- byos (bring your own server)
